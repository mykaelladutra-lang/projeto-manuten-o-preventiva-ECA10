# projeto-manutencao preventiva-ECA10
Projeto de desensolvimento de um programa para auxilio no setor de manutenção preventiva

A manutenção preventiva tem como objetivo evitar falhas inesperadas em equipamentos críticos. Neste projeto, o foco será o monitoramento de um motor elétrico industrial, analisando variáveis para identificar padrões que indiquem desgaste ou risco de falha de equipamentos presentes em uma fabrica.
